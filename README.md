## 実行方法
dotfilesLink.shはホームディレクトリにシンボリックリンクを貼る</br>
```
sh dotfilesLink.sh
```

## vimrc(dein)
### 構造
```
vim  
|  
|--- init ------ base.vim  
|             |- color.vim  
|             |- mapping.vim  
|             |- plugins.vim  
|
|--- plugins --- plugins-nerdtree-git-plugin.vim
              |- plugins-open-browser.vim
              |- plugins-vimshell.vim
```
